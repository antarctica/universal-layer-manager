# @ulm/leaflet changelog

All notable changes to `@ulm/leaflet` are documented in this file.

This project follows [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-19

### Added

- Initial Leaflet adapter for `@ulm/core`, including a factory for creating a Leaflet-backed layer manager.
- Type-safe bindings between Leaflet layers and the `@ulm/core` layer configuration.
